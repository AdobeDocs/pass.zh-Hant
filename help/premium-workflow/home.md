# [!UICONTROL Premium Workflow] 概覽

>[!IMPORTANT]
>
> 請務必隨時瞭解[產品公告](/help/authentication/product-announcements.md)頁面中彙總的最新Adobe Pass產品公告和淘汰時間表。

Adobe Pass的[!UICONTROL Premium Workflow]提供強大的工具套件，可改善驗證和串流可靠性。 透過[!UICONTROL Entitlement Service Monitoring]取得即時可見度，透過智慧型降級API確保運作時間，並透過[!UICONTROL TempPass]提供彈性存取。

增強[!UICONTROL Home-Based Authentication]和單一登入的使用者體驗，而MVPD狀態服務會主動追蹤端點效能並發出警示。 這些解決方案讓程式設計師和MVPD擁有可操作的深入分析、最佳化營運、提高參與度，並確保內容傳遞不中斷，即使在流量尖峰期亦然。

主要功能：

+ [Apple SSO概觀](sso-access/partner-sso/apple-sso/apple-sso-overview.md)
+ [Amazon SSO逐步指南(REST API V2)](sso-access/platform-sso/amazon-single-sign-on/amazon-sso-cookbook-rest-api-v2.md)
+ [Roku SSO逐步指南(REST API V2)](sso-access/platform-sso/roku-single-sign-on/roku-sso-cookbook-rest-api-v2.md)
+ [家用驗證(HBA)](hba-access/home-based-authentication.md)
+ [TempPass功能](temporary-access/temp-pass-feature.md)
+ [退化特徵](degraded-access/degradation-feature.md)
+ [軟體權利檔案服務監視概觀](/help/premium-workflow/esm/entitlement-service-monitoring-overview.md)
