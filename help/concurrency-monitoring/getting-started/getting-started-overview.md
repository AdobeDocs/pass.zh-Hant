# 開始使用並行監視 {#getting-started-overview}

歡迎使用並行監視！ 本指南將幫助您瞭解基礎知識，並讓您的整合快速啟動並執行。

## 並行監視解決了問題 {#problem-solved}

在現今的串流格局中，訂閱者期望能存取跨多部裝置和應用程式的內容。 不過，這對內容提供者來說是個挑戰：

- **內容授權合約**&#x200B;通常會限制同時串流的數量
- **頻寬成本**&#x200B;會隨著過多的同時使用而增加
- 當太多資料流降低效能時，**使用者體驗**&#x200B;會受影響
- **收入保護**&#x200B;需要防止帳戶共用濫用

並行監視提供這些挑戰的集中式解決方案。


## 並行監視的運作方式

### 核心功能

並行監視以&#x200B;**工作階段管理服務**&#x200B;的身分運作，該服務：

1. **即時追蹤您所有應用程式中的使用中串流工作階段**
2. **根據目前的使用模式評估企業原則**
3. 違反原則時&#x200B;**強制限制**
4. **發生衝突時提供解決選項**

## 不同利害關係人的優點

### 內容提供者（程式設計師）

- **保護內容權利**&#x200B;並遵守授權合約
- **防止過度使用，以最佳化頻寬成本**
- **改善使用者體驗**，在限制上提供明確的意見回饋
- **透過詳細報告取得使用狀況深入分析**

### 身分提供者(MVPD)

- **強制所有內容合作夥伴的訂閱者協定**
- **多個應用程式的集中式原則管理**
- **即時監視**&#x200B;使用模式
- **可擴充的架構**&#x200B;可處理數百萬個工作階段

### 一般使用者

- **清楚瞭解**&#x200B;使用量限制
- 所有應用程式中的&#x200B;**一致的體驗**
- **達到限制時解決衝突的選項**
- **透明意見反應**，說明限制存取的原因

## 服務註冊 {#service-registration}

若要開始使用並行監視，請連絡我們的[支援團隊](mailto:tve-support@adobe.com)，並提供下列資訊：

1. **公司名稱**&#x200B;和連絡人詳細資料
2. **您要與並行監視整合的應用程式**。 請為每個應用程式提供：
   1. 應用程式名稱
   2. 應用程式平台
3. **整合合作夥伴** (如果您是應另一方、程式設計師或MVPD的要求訂閱並行監視)
